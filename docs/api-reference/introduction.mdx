## Welcome to the Dysh API Reference

The Dysh API provides powerful endpoints for AI-powered recipe generation, user authentication, and profile management. All endpoints are RESTful and return JSON responses.

## Base URL

```
http://localhost:3000  # Development
https://api.dysh.app   # Production
```

## Authentication

Most endpoints require authentication using JWT tokens obtained through Apple Sign-In or Google Sign-In.

```bash
Authorization: Bearer <your_access_token>
```

## Response Format

All API responses follow a consistent format:

### Success Response
```json
{
  "data": {
    // Response data here
  },
  "status": "success"
}
```

### Error Response
```json
{
  "statusCode": 400,
  "message": "Error description",
  "error": "Bad Request"
}
```

## Rate Limits

- **Authentication endpoints**: 20 requests per minute
- **Recipe generation**: 10 requests per minute per user
- **User management**: 100 requests per hour per user
